JLArrangeController
======================

Rearrange icons, images or any views as you want.



## Usage

Copy below files into your project. 
```
JLArrangeController.h
JLArrangeController.m
JLArrangeView.h
JLArrangeView.m
```

There is a sample project named `DragMove`.
